# Bounty-execution mode

Use for bug, ML, security, benchmark, or challenge-style paid scopes where the output is a report, reproduction, evidence, or disclosure package.

Primary outputs in `preparation-packet.md`:
- scope/rules summary
- eligibility/payment/platform constraints
- safety/legal boundaries
- reproduction/report template
- evidence checklist
- disclosure/submission checklist
- risk blockers and stop conditions

Optional companion file:
- `application/submission-checklist.md`

Do not perform non-consensual testing, exploit systems, bypass access controls, or submit reports without Brayan's explicit approval and a safe scope.
