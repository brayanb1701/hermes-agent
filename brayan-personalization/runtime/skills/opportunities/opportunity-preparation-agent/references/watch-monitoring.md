# Watch-monitoring mode

Use for broad career pages, future cycles, generic portals, and high-interest organizations without a current specific application target.

Primary outputs in `preparation-packet.md` when selected:
- watch target summary
- relevant roles/programmes shortlist
- trigger conditions for future action
- preparation gaps
- suggested reminders/check dates
- decision needed before promotion to a more active workflow mode

Most watch records should stay `researched` until a specific target is chosen. Use `preparation-ready` for watch-monitoring only when Brayan explicitly wants a review/planning packet for that watch target.
