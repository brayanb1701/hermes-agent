# Scholarship-planning mode

Use for scholarships, international mobility programs, education funding, and programme portals where eligibility/documents/deadlines matter more than CV tailoring.

Primary outputs in `preparation-packet.md`:
- programme/call summary
- eligibility checklist
- shortlist workflow if the page is generic
- deadline table and uncertainty notes
- required document checklist
- blockers: transcript, passport, language proof, recommendations, translations, fees, accounts
- recommended next decisions and reminder points

Do not pretend a generic portal is a complete application. Keep status earlier unless a specific programme/call is actionable.
